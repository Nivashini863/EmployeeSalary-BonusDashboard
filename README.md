# Employee Salary & Bonus Dashboard

## Project Overview

This Power BI dashboard provides insights into employee salary, bonus, and department performance using interactive visualizations.

## Dashboard Features

- Total Employees
- Total Salary
- Average Salary
- Total Bonus
- Bonus Percentage
- Salary by Department
- Bonus by Department
- Salary by Company
- Employees by Employment Type

## Interactive Filters

- Department
- Company
- Employee Type
- Age

## Tools Used

- Power BI Desktop
- Microsoft Excel
- DAX
- Power Query
